1 Основное окно

В поле ввода необходимо ввести выражение, которое нужно посчитать. Максимальное количество символов - 255. Ввод символов осуществляется с клавиатуры или непосредственно в строке редактирования. Алгоритм главного окна блокирует большинство бессмысленных выражений, однако алгоритм вычислительной модели всё равно проводит дополнительную проверку строки ввода. Для очистки строки ввода используется кнопка "АС", удаление произвольной части строки можно осуществить вручную.

Результатом вывода может быть:

  1. число - выражение корректное, результат вычисления;
  2. "текст ошибки" - выражение некорректное, указывает в чем ошибка;
  3. nan - результат вычисления "нечисло";
  4. inf - результат вычисления плюс бесконечность;
  5. -inf - результат вычисления минус бесконечность.

1.1 Числа

  • * 0-9 - для записи чисел в поле ввода;
  • * x - кнопка "x" помещает переменную в основное поле ввода. Программа заменяет её на значение введённое в малое поле ввода справа от кнопки. Принимаются только значения с плавающей точкой двойной точности.
  • * e - числа в экспоненциальной записи имеют вид 6e+7 или 2e-3, калькулятор не воспринимает е в другом контекте и не будет возвращать его значение;

1.2 Арифметические операции

  • * Скобки - выражение, заключенное в скобки, считается более приоритетным;
  • * Сложение - сумма двух операндов;
  • * Вычитание - разность двух операндов;
  • * Умножение - произведение двух операндов ;
  • * Деление - частное двух операндов;
  • * Возведение в степень - первый операнд в степени второй операнд;

1.3 Функции

  • * cos(x) - вычисляет косинус числа x в радианах;
  • * sin(x) - вычисляет синус числа x в радианах;
  • * tan(x) - вычисляет тангенс числа x в радианах;
  • * acos(x) - вычисляет арккосинус числа x в радианах;
  • * asin(x) - вычисляет арксинус числа x в радианах;
  • * atan(x) - вычисляет арктангенс числа x в радианах;
  • * sqrt(x) - вычисляет квадратный корень из числа x;
  • * ln(x) - вычисляет натуральный логарифм числа x;
  • * log(x) - вычисляет десятичный логарифм числа x.
  • * mod(x) - вычисляет остаток от деления;

2 Построение графика функций по введенному выражению

Режим для построения графика функции открывается при нажатии кнопки "function graph". Для того чтобы построить график функции необходимо в основной строке ввести выражение с параметром x (значения для х задаются в области определения). При возникновении ошибок будет выведено сообщение с текстом ошибки. Если максимальное значение координаты меньше или равно минимальному, оно автоматически увеличивается на 1. При построении графика значения x введённые вручную будут игнорироваться.
